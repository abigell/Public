## Week 1 – History
- One writing system can represent many languages
- The Epic of Gilgamesh is one of the earliest texts

## Week 2 – Bits and Bytes
- A bit is the smallest unit of data
- A byte is made of 8 bits

## Week 3 – Number Systems
- Binary uses base 2
- Hexadecimal uses base 16

## Week 4 – Text Encoding
- ASCII represents characters as numbers
- Unicode supports many languages

## Week 5 – Markdown & HTML
- Markdown is easier to read than HTML
- HTML uses tags like <p> and <h1>
